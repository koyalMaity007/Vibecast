# 🌤️ Vibecast

## ✨ Overview
Vibecast is a modern weather app built to deliver clear, real-time weather updates with a smooth and intuitive experience. It focuses on simplicity, speed, and visually clean information so users can quickly understand the day ahead.

---

## 🌦️ Features

- Live weather updates based on location  
- “Feels like” temperature display  
- Clean and minimal UI design  
- Easy-to-read forecast layout  
- Lightweight and fast performance  
- Responsive design for all devices  

---

## 📸 Purpose

The goal of Vibecast is to make weather checking effortless and visually enjoyable, removing clutter and focusing only on what matters for your day.
